# Camera 5 — Entrance & Billing
Drop the recorded `.mp4` (or `.mov`) file here. Filename must start with
`YYYYMMDDHHMMSSmmm` (e.g. `20260608044448561_AH8174419_Barkerend - Kingz_16_video.mov`)
so the pipeline can parse the real recording-start time. Files are
gitignored (too large for GitHub), so each contributor brings their own
footage.
